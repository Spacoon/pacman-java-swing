# pacman-java-swing
This project for the GUI course at PJAIT is aimed at demonstrating basic knowledge of the Swing module in Java. It features a game with level selector as well as a leaderboard system.

![alt text](https://github.com/Spacoon/pacman-java-swing/blob/main/image.png)

# Keyboard controls
- Arrow Keys
  - Up Arrow: Move the character up
  - Down Arrow: Move the character down
  - Left Arrow: Move the character left
  - Right Arrow: Move the character right
- Escape (Esc)
  - Quit to the main menu
